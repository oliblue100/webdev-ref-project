HTML Reference Page

Use this page to refer back to if you need a refresher.

Sometimes when working on a project, its easy to forget how to use different features. Especially having to learn multiple languages like HTML, CSS, and Javascript. While they all work together, its hard to memorize everything. The sheer amount of features available is enourmous, its impossible to memorize everything. Even some basic features are hard to memorize. Personally, although I've been developing for quite some time, I still forget how to link a basic stylesheet. ("Its not 'url', its 'href!!!'") This is a somewhat of an HTML cheatsheet that you can use to refer back to incase I need to refresh my memory on how different features work.

Be sure to look at html source! There are comments there that provide more detailed information.
